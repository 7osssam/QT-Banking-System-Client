\doxysubsection{Detailed Description}
Provides an interface for administrative tasks within the banking system. 

The \mbox{\hyperlink{classAdminWidget}{Admin\+Widget}} class allows administrators to manage user accounts, view transaction history, and update settings. It includes tabs for managing the database, viewing transactions, and updating settings, as well as floating action buttons for user management tasks. The widget provides features for viewing database content, managing user accounts, and logging out of the application. The \mbox{\hyperlink{classAdminWidget}{Admin\+Widget}} communicates with the backend server through the \mbox{\hyperlink{classRequestManager}{Request\+Manager}} to perform various operations.

The widget consists of\+:
\begin{DoxyItemize}
\item Database tab\+: Displays database content and allows users.
\item Transactions tab\+: Shows transaction history for all bank accounts (from -\/\texorpdfstring{$>$}{>} to -\/\texorpdfstring{$>$}{>} amount).
\item Settings tab\+: Allows the admin to update their email address and password. 
\end{DoxyItemize}
