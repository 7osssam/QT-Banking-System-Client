\doxysubsection{Detailed Description}
A class for managing TCP client connections. 

The \mbox{\hyperlink{classTcpClient}{Tcp\+Client}} class is responsible for establishing a connection to a TCP server, sending requests, receiving responses, and handling various socket states and errors. It provides signals for connection status and incoming responses.
