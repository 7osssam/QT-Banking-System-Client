\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classUserWidget}{User\+Widget}} class provides a user interface for managing user accounts, performing transactions, and modifying settings within the banking application. 

This class represents the main user interface component that allows users to interact with their account information, view transaction history, perform transfers, and update personal settings. It includes several tabs for different functionalities such as Home, Transfer, and Settings.

The \mbox{\hyperlink{classUserWidget}{User\+Widget}} class manages the display of user data, handles user interactions, and communicates with the backend server through the \mbox{\hyperlink{classRequestManager}{Request\+Manager}}. It provides features for viewing transaction history, performing money transfers, updating account details, and logging out of the application.

The widget consists of\+:
\begin{DoxyItemize}
\item Home tab\+: Displays welcome message, current balance, and transaction history.
\item Transfer tab\+: Allows users to transfer funds to another account or email.
\item Settings tab\+: Provides options to update email, update password, and log out. 
\end{DoxyItemize}
