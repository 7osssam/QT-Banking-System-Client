\doxysubsection{Detailed Description}
A class for managing client requests and responses. 

The \mbox{\hyperlink{classClientHandler}{Client\+Handler}} class handles the client-\/side operations such as sending requests, receiving responses, and managing connections. It uses the \mbox{\hyperlink{classTcpClient}{Tcp\+Client}} class to handle the actual TCP communication.
