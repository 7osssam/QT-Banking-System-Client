 \doxysection*{Overview}

QT-\/\+Banking-\/\+System-\/\+Client is the client-\/side implementation of a banking system, built using the Qt 6/\+C++ framework. The application provides a graphical user interface (GUI) for users and administrators to manage bank accounts, process transactions, and handle various banking operations. The client communicates with a server-\/side application through a TCP socket and handles database operations using Postgre\+SQL with Supabase hosting.

\doxysection*{Features}


\begin{DoxyItemize}
\item {\bfseries{User Management\+:}} Administrators can manage user accounts, including creating, updating, and deleting accounts.
\item {\bfseries{Transaction Management\+:}} Users can view transaction history and process transactions securely.
\item {\bfseries{Real-\/\+Time Updates\+:}} The client receives real-\/time updates from the server regarding account balances and transaction statuses.
\item {\bfseries{Secure Communication\+:}} Communication between the client and server is secured with appropriate security measures (e.\+g., hashing, encryption).
\item {\bfseries{Responsive UI\+:}} The application features a responsive and user-\/friendly interface, utilizing Qt\textquotesingle{}s powerful GUI capabilities with the help of \href{https://github.com/sebcaux/qt-material-widgets}{\texttt{ qt-\/material-\/widgets}} library for a modern look and feel.
\end{DoxyItemize}
